# ThreatLens

> SOC Automation & IOC Enrichment Platform

ThreatLens is a SOC automation platform that streamlines the investigation of Indicators of Compromise (IOCs) by automatically collecting, correlating, and analyzing threat intelligence from multiple sources. The platform enriches IP addresses, domains, URLs, and file hashes using public threat intelligence feeds, assigns a confidence-based risk score, maps findings to the MITRE ATT&CK framework, and generates professional investigation reports.

---

## 🚀 Features (Planned)

- IOC Validation
- VirusTotal Integration
- AbuseIPDB Integration
- AlienVault OTX Integration
- URLHaus Integration
- Confidence-based Risk Scoring
- MITRE ATT&CK Mapping
- HTML/PDF Report Generation
- Investigation History
- Wazuh SIEM Integration
- Bulk IOC Analysis
- Interactive Dashboard
